Liste des erreurs

Lorsque vous utiliser mal l'API ou qu'elle rencontre un problème, elle vous donne un message d'erreur ainsi qu'un code erreur.

CodeDescription
0Vous avez effectué trop de requêtes.
1Vous avez effectué trop de requêtes.
2Vous avez effectué trop de requêtes.
3Vous avez effectué trop de requêtes.
4Vous n'avez pas inclus d'UUID dans votre requête (?uuid=<votre UUID>).
5L'API https://johanstickman.com a rencontré une erreur inconnue.
6Impossible d'obtenir les informations de votre compte.
7Votre compte est banni. Plus d'informations via https://johanstickman.com/account
8Vous n'avez pas inclus le texte de votre message dans votre requête (?message=<votre message>).
9La taille de votre message doit dépasser les 2 caractères.
10La taille de votre message doit faire moins de 1000 caractères.
