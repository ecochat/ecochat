Liste des message

Les 50 derniers messages du chat vous seront renvoyé avec le pseudo, l'id et le message de l'auteur et un séparateur pour chaque élément (⁜).

GET /api/chat/list?token=....#

Résultat :

{  "message": "Liste des messages.",  "error": false,  "shortError": false,  "messages": [    "Johan_stickman⁜356030913075200743⁜Hey !",    "Quelque_un⁜039194141298038145⁜Ecochat > Terminalchat",    "Pseudo⁜ID⁜Message",  ]}

Rate limit#

85 requêtes maximum toute les 20 secondes

Résultat :

{  "message": "En raison d'un nombre de requête trop élevé, vous ne pouvez plus voir les messages pour les 10 prochaines secondes.",  "error": true,  "shortError": "rateLimitedChatList"}
